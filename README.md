A full-stack Food Delivery Web Application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). This app allows users to browse food items, add them to cart, and place orders.

🧑 User Authentication (Login / Signup)
🍕 Browse Food Items
🛒 Add to Cart
💳 Order Placement
📦 Order Management
🔐 Secure API with JWT
⚡ Fast and Responsive UI
